```go
package bigquery

import (
    "fmt"
    "strings"
    "charm.land/lipgloss/v2"
    "charm.land/lipgloss/v2/table"
)

// RenderQueryResults renders query results as a styled Lipgloss table.
// Returns both a Display string (styled for TUI) and a Content string (plain text for LLM).
func RenderQueryResults(headers []string, rows [][]string, totalRows uint64, maxDisplayRows int, cost float64, durationMs int64, bytesScanned int64) (display string, content string)

// RenderTableDetail renders a table's schema as formatted text.
func RenderTableDetail(detail *schema.TableDetail) (display string, content string)

// RenderDatasetList renders a list of datasets.
func RenderDatasetList(datasets []schema.DatasetInfo, projectID string) (display string, content string)

// RenderColumnSearch renders column search results.
func RenderColumnSearch(results []schema.ColumnSearchResult, query string) (display string, content string)

// FormatBytes formats byte count as human-readable (KB, MB, GB, TB).
func FormatBytes(bytes int64) string

// FormatCost formats dollar amount.
func FormatCost(cost float64) string

// FormatDuration formats milliseconds as "X.Xs" or "Xm Xs".
func FormatDuration(ms int64) string
```

RenderQueryResults implementation:
- Create `table.New()` with `Border(lipgloss.RoundedBorder())`
- Set `BorderStyle` foreground to `lipgloss.Color("#374151")` (inputBorderColor equivalent, hardcoded since render.go is not in tui package)
- Set `Headers(headers...)` with bold + accent color styling
- Truncate rows to maxDisplayRows
- Handle NULL values: render "NULL" in italic
- Calculate column widths: min 8, max 48, numeric columns right-aligned, total width capped at 120 chars
- Render table with `t.Render()`
- Add overflow indicator: `"  {N} more rows"` if totalRows > displayed
- Add cost footer: `"  {cost} | {duration} | {bytes} scanned"`
- Content (for LLM): plain text table with headers and rows (no ANSI), truncated at 50 rows, with cost/footer in plain text
- Display (for TUI): styled Lipgloss table with ANSI codes

FormatBytes:
- < 1024 -> "{n} B"
- < 1024*1024 -> "{n/1024:.1f} KB"
- < 1024*1024*1024 -> "{n/(1024^2):.1f} MB"
- < 1024^4 -> "{n/(1024^3):.1f} GB"
- else -> "{n/(1024^4):.1f} TB"

FormatDuration:
- < 1000ms -> "{ms}ms"
- < 60000ms -> "{s:.1f}s"
- else -> "{m}m {s}s"

RenderTableDetail: per UI-SPEC:
- Header: `"  {dataset}.{table}"` in bright+bold
- Metadata: `"  Type: {type} | Rows: {rowcount} | Size: {size}"`
- Partition: `"  Partitioned by: {field} ({type}) | Clustered by: {fields}"`
- Columns table using Lipgloss table with headers: Column Name, Type, Nullable, Description
- Partition key annotated as `"(partition key)"` in description
- Clustering annotated as `"(cluster N)"`

RenderDatasetList: per UI-SPEC:
- Header: `"  Datasets in project {projectID}:"`
- Each dataset: `"    {name}  {tableCount} tables  {size}"`

**2. Create BigQueryQuery tool (internal/tools/bigquery/query_tool.go):**

```go
package bigquery

type QueryTool struct {
    client      *bq.Client
    cache       *schema.Cache
    costTracker *bq.CostTracker
    costConfig  *config.CostConfig
}

func NewQueryTool(client *bq.Client, cache *schema.Cache, costTracker *bq.CostTracker, costConfig *config.CostConfig) *QueryTool
```

Implement Tool interface:
- `Name()`: return `"bigquery_query"`
- `Description()`: return `"Execute a BigQuery SQL query. Use this to run SELECT queries, DML statements, or DDL operations against BigQuery. The query will be cost-estimated before execution."`
- `InputSchema()`: return JSON Schema:
  ```json
  {
    "type": "object",
    "properties": {
      "sql": {"type": "string", "description": "The SQL query to execute"},
      "dry_run": {"type": "boolean", "description": "If true, only estimate cost without executing"}
    },
    "required": ["sql"]
  }
  ```
- `RiskLevel()`: return `permission.RiskDestructive` (worst case; agent loop overrides dynamically)
- `Execute(ctx, input)`:
  1. Parse input: `{sql: string, dry_run?: bool}`
  2. If dry_run is true, delegate to cost estimation only (return cost estimate in Result)
  3. Run dry-run first: `client.EstimateCost(ctx, sql)`
  4. If cost estimation returns -1 (DML), set costDisplay to "Cost: cannot estimate for DML (syntax valid)"
  5. If estimatedCost > costConfig.MaxQueryCost, return Result with Content: "Query blocked: estimated cost ${cost} exceeds maximum ${max}. Add partition filters or increase cost.max_query_cost_usd in config." and IsError: true
  6. Execute query: `client.ExecuteQuery(ctx, sql, costConfig.MaxDisplayRows)`
  7. Track cost: `costTracker.Record(QueryCostEntry{SQL: sql, BytesScanned: bytes, Cost: cost, DurationMs: duration})`
  8. Render results: `RenderQueryResults(headers, rows, totalRows, maxDisplayRows, cost, duration, bytes)`
  9. Auto-invalidate cache after DDL: if ClassifySQLRisk(sql) == RiskDDL or RiskDestructive, call cache.InvalidateTable for affected table (parse table name from SQL -- simple regex for DROP/CREATE/ALTER TABLE `dataset.table`)
  10. Return `&tools.Result{Content: content, Display: display}`

```go
package bigquery

type SchemaTool struct {
    cache *schema.Cache
}

func NewSchemaTool(cache *schema.Cache) *SchemaTool
```

Implement Tool interface:
- `Name()`: return `"bigquery_schema"`
- `Description()`: return `"Explore BigQuery schema: list datasets, list tables in a dataset, describe a table's columns/partitioning/clustering, or search for columns by name across all cached tables."`
- `InputSchema()`: return JSON Schema:
  ```json
  {
    "type": "object",
    "properties": {
      "action": {
        "type": "string",
        "enum": ["list_datasets", "list_tables", "describe_table", "search_columns"],
        "description": "The schema action to perform"
      },
      "dataset": {
        "type": "string",
        "description": "Dataset ID (required for list_tables and describe_table)"
      },
      "table": {
        "type": "string",
        "description": "Table ID (required for describe_table)"
      },
      "query": {
        "type": "string",
        "description": "Search query for search_columns action"
      }
    },
    "required": ["action"]
  }
  ```
- `RiskLevel()`: return `permission.RiskReadOnly` (always read-only)
- `Execute(ctx, input)`:
  1. Parse input: `{action, dataset?, table?, query?}`
  2. Switch on action:
     - `"list_datasets"`: call cache.GetDatasets(), render with RenderDatasetList
     - `"list_tables"`: validate dataset provided, call cache.GetTables(dataset), render as table list
     - `"describe_table"`: validate dataset+table, call cache.GetTableDetail(dataset, table), render with RenderTableDetail
     - `"search_columns"`: validate query provided, call cache.SearchColumns(query, 20), render with RenderColumnSearch
  3. If cache is not populated (cache.IsPopulated() == false), return Result with Content: "No schema cache found. Ask a question about your data or run /sync to build the cache." and IsError: false
  4. Return `&tools.Result{Content: content, Display: display}`

- `Description()`: return `"Estimate the cost of a BigQuery SQL query without executing it. Returns bytes to be scanned and estimated dollar cost based on on-demand pricing."`

  4. If cost == -1 (DML), return "Cost: cannot estimate for DML (syntax valid)"
  5. Format result: "Estimated cost: ${cost} ({bytes})"
  6. Content includes the estimate text. Display includes cost coloring (but since this is a tool, just return plain text -- TUI handles the styling via ToolEndEvent)
  7. Return `&tools.Result{Content: content, Display: display}`
