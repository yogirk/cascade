**3. Create BQ client wrapper (internal/bigquery/client.go):**

```go
package bigquery

import (
    "context"
    "fmt"
    "cloud.google.com/go/bigquery"
    "golang.org/x/oauth2"
    "google.golang.org/api/iterator"
    "google.golang.org/api/option"
)

type Client struct {
    bq         *bigquery.Client
    projectID  string
    location   string
    pricePerTB float64
}

func NewClient(ctx context.Context, projectID, location string, ts oauth2.TokenSource, pricePerTB float64) (*Client, error)
func (c *Client) ExecuteQuery(ctx context.Context, sql string, maxRows int) (headers []string, rows [][]string, totalRows uint64, schema bigquery.Schema, err error)
func (c *Client) EstimateCost(ctx context.Context, sql string) (bytesProcessed int64, estimatedCost float64, err error)
func (c *Client) RunQuery(ctx context.Context, sql string) (*bigquery.RowIterator, error)
func (c *Client) ProjectID() string
func (c *Client) Close() error
```

Key implementation details:
- `NewClient`: Create `bigquery.NewClient(ctx, projectID, option.WithTokenSource(ts))`, set `bq.Location = location`, store `pricePerTB` (default 6.25)
- `ExecuteQuery`: Run `c.bq.Query(sql)`, iterate with `it.Next(&row)` up to `maxRows`, convert `[]bigquery.Value` to `[]string` via defensive type switching (handle nil as "NULL", nested structs as JSON, arrays as comma-separated). Return headers from `it.Schema` field names. Return `it.TotalRows` for total count.
- `EstimateCost`: Set `q.DryRun = true`, call `q.Run(ctx)`, read `job.LastStatus().Statistics.TotalBytesProcessed`. Calculate cost as `float64(bytes) / 1e12 * c.pricePerTB`. For DML (bytes == 0 when TotalBytesProcessed returns 0 for DML), return -1 for cost to signal "cannot estimate".
- `RunQuery`: Lower-level method returning raw RowIterator for INFORMATION_SCHEMA queries in populate.go
- Handle `bigquery.Value` types defensively: nil -> "NULL", string/int/float -> fmt.Sprintf, time.Time -> RFC3339, []Value -> JSON array, nested -> JSON object

**1. Create SQLite migrations (internal/schema/migrations.go):**

```go
package schema

const schemaSQL = `
CREATE TABLE IF NOT EXISTS datasets (
    dataset_id TEXT PRIMARY KEY,
    project_id TEXT NOT NULL,
    location TEXT,
    description TEXT,
    labels TEXT,
    last_refreshed INTEGER NOT NULL
);

CREATE TABLE IF NOT EXISTS tables (
    dataset_id TEXT NOT NULL,
    table_id TEXT NOT NULL,
    table_type TEXT,
    description TEXT,
    row_count INTEGER,
    size_bytes INTEGER,
    partition_field TEXT,
    partition_type TEXT,
    clustering_fields TEXT,
    labels TEXT,
    last_modified INTEGER,
    last_refreshed INTEGER NOT NULL,
    PRIMARY KEY (dataset_id, table_id)
);

CREATE TABLE IF NOT EXISTS columns (
    dataset_id TEXT NOT NULL,
    table_id TEXT NOT NULL,
    column_name TEXT NOT NULL,
    data_type TEXT NOT NULL,
    is_nullable INTEGER,
    description TEXT,
    ordinal_position INTEGER,
    is_partitioning INTEGER DEFAULT 0,
    clustering_ordinal INTEGER,
    PRIMARY KEY (dataset_id, table_id, column_name)
);

CREATE VIRTUAL TABLE IF NOT EXISTS schema_fts USING fts5(
    dataset_id,
    table_id,
    column_name,
    description,
    content=''
);

CREATE INDEX IF NOT EXISTS idx_columns_table ON columns(dataset_id, table_id);
`

func migrate(db *sql.DB) error {
    _, err := db.Exec(schemaSQL)
    return err
}
```

**2. Create schema cache manager (internal/schema/cache.go):**

```go
package schema

import (
    "database/sql"
    "os"
    "path/filepath"
    "sync"
    _ "modernc.org/sqlite"
)

type Cache struct {
    mu       sync.RWMutex
    db       *sql.DB
    cacheDir string
}

func NewCache(cacheDir string) *Cache
func (c *Cache) Open(projectID string) error       // Opens/creates DB at cacheDir/projectID.db, runs migrate, sets WAL+busy_timeout
func (c *Cache) Close() error
func (c *Cache) IsPopulated() bool                  // Returns true if tables table has any rows
func (c *Cache) GetDatasets() ([]DatasetInfo, error)
func (c *Cache) GetTables(datasetID string) ([]TableInfo, error)
func (c *Cache) GetTableDetail(datasetID, tableID string) (*TableDetail, error)  // Full table metadata + columns
func (c *Cache) InvalidateTable(datasetID, tableID string) error  // Delete from tables + columns + FTS after DDL
func (c *Cache) DB() *sql.DB                        // Expose for populate and search
```

Types:
```go
type DatasetInfo struct {
    DatasetID   string
    ProjectID   string
    Location    string
    Description string
    TableCount  int
    TotalBytes  int64
}

type TableInfo struct {
    DatasetID       string
    TableID         string
    TableType       string
    Description     string
    RowCount        int64
    SizeBytes       int64
    PartitionField  string
    ClusteringFields []string
}

type TableDetail struct {
    TableInfo
    Columns []ColumnInfo
}

type ColumnInfo struct {
    Name              string
    DataType          string
    IsNullable        bool
    Description       string
    OrdinalPosition   int
    IsPartitioning    bool
    ClusteringOrdinal int
}
```

Open() implementation:
- `os.MkdirAll(cacheDir, 0o755)`
- `dbPath := filepath.Join(cacheDir, projectID+".db")`
- `sql.Open("sqlite", dbPath)`
- `db.Exec("PRAGMA journal_mode=WAL")`
- `db.Exec("PRAGMA busy_timeout=5000")`
- `migrate(db)`

InvalidateTable():
- Delete from `columns WHERE dataset_id=? AND table_id=?`
- Delete from `tables WHERE dataset_id=? AND table_id=?`
- Delete from `schema_fts WHERE dataset_id=? AND table_id=?`

GetTableDetail():
- Query `tables` row for metadata
- Query `columns WHERE dataset_id=? AND table_id=? ORDER BY ordinal_position`
- Parse `clustering_fields` JSON array from tables row
- Build TableDetail with all ColumnInfo

**3. Create INFORMATION_SCHEMA populator (internal/schema/populate.go):**

```go
package schema

type Populator struct {
    cache *Cache
    bq    BQQuerier  // interface: RunQuery(ctx, sql) -> rows
}

type BQQuerier interface {
    RunQuery(ctx context.Context, sql string) (RowIterator, error)
    ProjectID() string
}

type ProgressFunc func(completed, total int)

func NewPopulator(cache *Cache, bq BQQuerier) *Populator
func (p *Populator) PopulateDataset(ctx context.Context, datasetID string, progress ProgressFunc) error
func (p *Populator) PopulateAll(ctx context.Context, datasets []string, progress ProgressFunc) error
```

PopulateDataset():
1. Run `SELECT table_name, table_type, ... FROM \`{project}.{dataset}.INFORMATION_SCHEMA.TABLES\` t LEFT JOIN \`{project}.{dataset}.INFORMATION_SCHEMA.TABLE_OPTIONS\` opt ON t.table_name = opt.table_name AND opt.option_name = 'description'`
2. Run `SELECT table_name, column_name, ordinal_position, is_nullable, data_type, column_description, is_partitioning_column, clustering_ordinal_position FROM \`{project}.{dataset}.INFORMATION_SCHEMA.COLUMNS\` ORDER BY table_name, ordinal_position`
3. Run `SELECT table_name, total_rows, total_logical_bytes FROM \`{project}.{dataset}.INFORMATION_SCHEMA.TABLE_STORAGE\`` (wrap in error handling -- may not be available for all table types)
4. Insert/upsert into cache tables and columns within a transaction
5. For each table+column, also INSERT into schema_fts (dataset_id, table_id, column_name, description)
6. Call progress callback with completed/total table counts

PopulateAll():
- Loop over datasets, call PopulateDataset for each
- Accumulate total table count across datasets for progress
