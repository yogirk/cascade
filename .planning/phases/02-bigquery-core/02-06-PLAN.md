**In `internal/app/bigquery.go`:**

1. Change `EnsureCachePopulated` to accept an `onCacheReady func()` callback:
```go
func (c *BigQueryComponents) EnsureCachePopulated(ctx context.Context, datasets []string, events chan types.Event, onCacheReady func()) {
	if c == nil || c.Cache.IsPopulated() || len(datasets) == 0 {
		return
	}
	go func() {
		err := c.Populator.PopulateAll(ctx, datasets, nil)
		if err != nil && events != nil {
			events <- &types.ErrorEvent{Err: fmt.Errorf("lazy cache build failed: %w", err)}
			return
		}
		if err == nil && onCacheReady != nil {
			onCacheReady()
		}
	}()
}
```

Key changes from existing code:
- Added `onCacheReady func()` parameter
- Added early `return` after error event emission (don't call onCacheReady on failure)
- Call `onCacheReady()` on successful population
